# 🎂 Shopecake

Bienvenue sur **Shopecake** - Une plateforme de vente en ligne de délicieux gâteaux !

## 📝 Description

Shopecake est une application web moderne qui permet aux clients de découvrir, commander et se faire livrer des gâteaux faits maison de qualité. Notre mission est de rendre les gâteaux artisanaux accessibles à tous !

## ✨ Fonctionnalités

- 🛍️ **Catalogue de produits** - Découvrez notre sélection de gâteaux
- 🛒 **Panier d'achat** - Ajoutez vos gâteaux préférés facilement
- 💳 **Paiement sécurisé** - Achetez en toute confiance
- 🚚 **Livraison** - Recevez vos gâteaux à domicile
- 👤 **Compte utilisateur** - Créez votre profil et suivez vos commandes
- ⭐ **Avis clients** - Partagez votre expérience
